Обертка над тегом <img>, основное назначение - загрузка картинок по URL. 

Dino.widget({
  dtype: 'image',
  imageUrl: 'img/stegosaurus_128x128.png',
});

Простая картинка предопределенного размера (как правило - 16x16), полностью управляется стилями. По умолчанию иконка имеет абсолютное позиционирование, т.к. чаще всего является декорационным элементом сложного виджета. 

Dino.widget({
  dtype: 'box',
  layout: 'row-layout',
  defaultItem: {
    dtype: 'icon',
    style: {'position': 'relative'}, //<-- для того, чтобы компоновка смогла определить размеры компонентов
    baseCls: 'ui-icon'
  },
  items: [
    { cls: 'ui-icon-newwin' },
    { cls: 'ui-icon-refresh' },
    { cls: 'ui-icon-shuffle' },
    { cls: 'ui-icon-folder-open' },
    { cls: 'ui-icon-document' },
    { cls: 'ui-icon-note' }
  ]
});

Поздняя загрузка позволяет перед добавлением изображения в разметку узнать его размеры, а значит появляется возможность выполнить пропорциональное масштабирование. 

Dino.object({
  dtype: 'box',
  items: [{
    // обычная загрузка
    dtype: 'image',
    imageUrl: 'img/Triceratops_128x128.png',
    width: 60,   
    height: 100   
  }, {
    // поздняя загрузка
    dtype: 'async-image',
    imageUrl: 'img/Triceratops_128x128.png',
    width: 60,
    height: 100   
  }]
});
